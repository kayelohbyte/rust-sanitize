**This project does not accept merge requests on GitLab.**

This GitLab project is an automated push mirror of the canonical repository
on GitHub. The mirror force-pushes GitHub state, so anything merged here
would be silently overwritten on the next sync.

Please open your change as a pull request on GitHub instead:
https://github.com/kayelohbyte/scour-secrets/pulls

(Issues, on the other hand, are welcome on either platform.)
